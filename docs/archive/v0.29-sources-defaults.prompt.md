## ⚠️ The trap that will break this (handoff §8)

`_builtin_log_sources()` returns **resolved absolute paths for the current machine** —
`Path.home()` is baked in, and env overrides (`CLAUDE_CONFIG_DIR`, `KIRO_DATA_DIR`, …)
redirect them. If you `str(path)` those into the block, docgen output differs per developer
and `--check` fails in CI.

The block must emit the **`~`-relative default** form, generated **independently of the
current env**. Solve this deliberately and test it, or the feature is broken on arrival.
