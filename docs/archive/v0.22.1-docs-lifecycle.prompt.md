exploring the repo can't tell live spec from historical build instruction
(e.g. the cost-impact handoff still sketches S5 as "3+3 tasks"; the shipped
gate is `MIN_COMPARE_N=5`). History is valuable — as an *archive with a
narrative spine*, not as loose files beside the design of record.
